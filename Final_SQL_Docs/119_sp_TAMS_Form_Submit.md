# **TAMS Admin – Submit TAR Form**

## Purpose
Submits a Track Access Request (TAR) by inserting or updating sector, station, power‑sector, and attachment records, determining the appropriate workflow, generating a reference number, and, for urgent requests, notifying the Head of Department (HOD) via email.
